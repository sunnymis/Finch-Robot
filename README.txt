A Finch robot is a small robot made at Carnegie Mellon's CREATE Lab
More information can be found at www.finchrobot.com

In this project we were able to make the Finch do many different things
It is able to:
Traverse the perimeter of the map
Draw the map in real-time 
Find the brightest spot
Find the hottest spot

The file User.txt in Team h Finch/src/instructionsandnotes shows the client
how to run the project
